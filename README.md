# Customer Retention & Sales Analysis (SQL + Power BI)
## `Project Overview:`

This project analyzes customer behavior, retention patterns, and sales performance using SQL and Power BI.

### The goal is to identify key business insights such as:

Customer retention trends

High-value customers

Sales performance across regions/products

Profitability and discount impact

### 🔹 Tools & Technologies
SQL (Data Cleaning & Analysis)

Power BI (Dashboard & Visualization)

Excel / CSV Dataset

### 🔹 Dataset Description
The dataset contains:

Customer details

Order transactions

Sales, profit, and discount data

Product and regional information

### 🔹 Key SQL Operations
Data cleaning (handling nulls, duplicates)

Aggregations (SUM, AVG, COUNT)

Joins and filtering

Customer segmentation queries

### 🔹 Power BI Dashboard Features
Sales Performance Overview

Customer Segmentation

Profit vs Discount Analysis

Region-wise Sales Insights

Retention & Repeat Customers

### 🔹 Key Insights
Identified high-value customers contributing to majority of revenue

Found regions with high sales but low profit margins

Observed negative impact of excessive discounts on profitability

Highlighted repeat customer patterns for retention strategies

### 🔹 Project Workflow
Data collection

Data cleaning using SQL

Exploratory data analysis

Data visualization using Power BI

Business insights generation
